CrossWord Puzzle Game

Project Description

A feature-rich crossword puzzle game built with C++, Object-Oriented Programming principles, and Qt GUI framework.
Test your vocabulary and puzzle-solving skills across three difficulty levels with increasing challenges!

Why CrossWord?

  • Three Difficulty Levels: Easy, Medium, Hard
  • Interactive Gameplay
  • Score Tracking
  • Hint System
  • Timer Functionality
  • Responsive Design

Features

  • Three Difficulty Levels:

    • Easy: 5 questions
    • Intermediate: 7 questions
    • Hard: 9 questions

  • Interactive GUI with Qt framework

  • Hint System: Get assistance at the cost of 2 points

  • Scoring System:

    • Correct answer: +5 points
    • Wrong answer: -3 points
    • Question skip: 0 points

  • Time Bonus: Complete puzzles within time limits for extra points

  • Static Puzzles: Carefully crafted grids and questions for each level

How To Play

  • Select your preferred difficulty level.
  • Read the clues and type your answers in the grid.
  • Use the hint button if you're stuck (costs 2 points).
  • Complete the puzzle as quickly as possible to earn time bonuses.
  • Skip questions if needed (no points awarded).
  • Submit your answers to see your score and time taken.
  • Try to complete the puzzle as quickly and accurately as possible!

Scoring System

  • Base points for correct answers: +5
  • Penalty for incorrect answers: -3
  • Hint cost: -2 points per use
  • Time bonuses awarded for quick completion
  • Final score is calculated based on accuracy, hints used, and completion time

Technologies Used

  • C++
  • Qt Framework
  • Object-Oriented Programming (OOP)
  • Custom game engine for puzzle management and scoring
  • Data Structures:
    • Arrays for grid representation
    • Hash Table
    • Maps for clue-to-answer mapping
